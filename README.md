## Tomath
“Tomath” merupakan aplikasi bank soal gamifikasi yang dibangun untuk platform mobile. Dalam pengembangannya, bank soal pada aplikasi tomath hanya terbatas pada pelajaran matematika tingkat SMP Kelas 7. Aplikasi ini  tidak memerlukan akses internet, sehingga pengguna dapat menggunakan aplikasi dalam mode offline.
<br><br>
aplikasi “Tomath” secara eksklusif berfokus pada platform Android, tanpa lingkup untuk pengembangan di iOS atau platform lain.
